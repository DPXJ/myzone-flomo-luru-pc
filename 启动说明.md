# 🚀 Flomo快速记录 - 启动说明

## 方式一：使用批处理文件（推荐）

直接双击 `run.bat` 文件即可启动程序。

该脚本会自动：
- ✅ 检查Python环境
- ✅ 检查并安装依赖
- ✅ 启动应用程序

## 方式二：命令行启动

### 1. 安装依赖（首次运行）

```bash
pip install -r requirements.txt
```

### 2. 启动程序

```bash
python main.py
```

## 首次配置

1. 程序启动后会在系统托盘显示图标
2. 右键托盘图标 → 选择「⚙️ 设置」
3. 填入你的Flomo API地址（在Flomo网页版「设置→API」获取）
4. 点击「测试连接」验证
5. 保存设置

## 开始使用

配置完成后：

1. 按 `Ctrl+Alt+F` 唤起输入窗口
2. 输入内容和标签
3. 按 `Ctrl+Enter` 或点击「发送」
4. 笔记自动同步到Flomo ✅

---

**重要提示：** 
- 如果快捷键不生效，请尝试以**管理员权限**运行程序
- Windows系统某些情况下需要管理员权限才能注册全局热键

## 系统要求

- Windows 10/11
- Python 3.10 或更高版本
- 网络连接
