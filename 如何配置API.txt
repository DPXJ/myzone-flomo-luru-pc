          🔑 如何配置 Flomo API 地址
═══════════════════════════════════════════════════════

📌 第一步：获取你的 Flomo API 地址

1. 打开浏览器，访问：https://flomoapp.com/

2. 登录你的 Flomo 账户

3. 点击右上角头像 → 设置 → API

4. 复制显示的 API 地址（格式类似：）
   https://flomoapp.com/iwh/MxxxxxxxxxxxxxxxxQ/

───────────────────────────────────────────────────────

📌 第二步：在应用中配置 API

方法1（推荐）：
  1. 双击运行「后台启动.bat」
  2. 在系统托盘（屏幕右下角）找到应用图标
  3. 右键点击图标
  4. 选择「⚙️ 设置」
  5. 在「API地址」框中粘贴你复制的地址
  6. 点击「测试连接」按钮
  7. 看到"测试成功"后，点击「保存」

方法2（如果看不到托盘图标）：
  1. 按快捷键 Ctrl+Alt+F
  2. 如果弹出输入窗口，说明程序在运行
  3. 看系统托盘右下角（可能在隐藏图标中）
  4. 找到应用图标后右键 → 设置

───────────────────────────────────────────────────────

📌 第三步：测试是否配置成功

1. 按快捷键：Ctrl+Alt+F

2. 输入测试内容：这是我的第一条笔记

3. 按 Ctrl+Enter 发送

4. 如果提示"发送成功"，说明配置正确！

5. 打开 Flomo 手机App 或网页版，应该能看到这条笔记

───────────────────────────────────────────────────────

⚠️ 常见问题

Q1: 找不到系统托盘图标？
A: 点击系统托盘的「^」展开隐藏图标

Q2: API 地址在哪里复制？
A: Flomo网页版 → 头像 → 设置 → API

Q3: 提示"API地址无效"？
A: 检查是否复制了完整地址，包括 https:// 开头

Q4: 关闭命令行窗口后程序退出了？
A: 使用「后台启动.bat」而不是其他启动脚本

Q5: 按快捷键没反应？
A: 1) 检查程序是否在运行（看系统托盘）
   2) 尝试以管理员身份运行
   3) 在设置中更换其他快捷键

───────────────────────────────────────────────────────

✨ 配置完成后的使用流程

1. 双击「后台启动.bat」（只需启动一次）
2. 按 Ctrl+Alt+F 唤起输入窗口
3. 输入内容和标签
4. 按 Ctrl+Enter 发送
5. 自动同步到 Flomo ✅

───────────────────────────────────────────────────────

📞 需要帮助？

如果配置遇到问题：
1. 检查网络连接是否正常
2. 确认 Flomo 账户可以正常登录
3. 重新获取 API 地址
4. 使用「一键修复.bat」重新安装依赖
