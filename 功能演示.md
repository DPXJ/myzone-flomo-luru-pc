## 💡 使用技巧

### 技巧1: 快捷键技巧

```
Ctrl+Alt+F    → 唤起输入窗口
Ctrl+Enter    → 快速发送（无需鼠标）
ESC           → 快速关闭（不保存）

组合使用: 
  Ctrl+Alt+F → 输入内容 → Ctrl+Enter
  全程无需鼠标，效率最高！
```

### 技巧2: 标签技巧

```
✅ 推荐做法:
  灵感 想法         → 自动转为 #灵感 #想法
  Python 学习       → 自动转为 #Python #学习

❌ 不推荐做法:
  #灵感#想法        → 虽然也可以，但不够简洁
```

### 技巧3: 默认标签

```
场景: 专门用于工作记录
设置: 默认标签设为 "工作 待办"
效果: 每次打开都自动填充这两个标签
优势: 省去重复输入
```

### 技巧4: 窗口大小

```
技巧: 调整窗口大小后，会自动记住
方法: 拖动窗口边缘调整
效果: 下次打开保持相同大小
```
