✅ 程序确实已经启动了！

但是，Flomo快速记录是一个**系统托盘应用**，
它不会出现在任务栏上，而是在系统托盘（右下角）运行。

═══════════════════════════════════════════════════════════════

🔍 如何找到应用图标：

   【第1步】找到系统托盘区域
   
      → 看电脑屏幕的 **右下角**
      → 找到时间和日期显示的旁边
      → 那里有一排小图标
   
   【第2步】展开隐藏的图标
   
      → 如果图标较多，可能会被隐藏
      → 点击 **"^"** 或 **"∧"** 符号
      → 这会展开所有隐藏的托盘图标
   
   【第3步】找到 Flomo 图标
   
      → 在展开的图标中找 "Flomo快速记录"
      → 可能显示为一个应用程序图标

═══════════════════════════════════════════════════════════════

🖱️ 找到图标后如何使用：

   • **右键点击图标** → 打开菜单
     
     菜单选项:
     - 📝 打开输入窗口 (Ctrl+Alt+F)
     - ⚙️ 设置
     - ℹ️ 关于
     - 🚪 退出

   • **双击图标** → 直接打开输入窗口

   • **按快捷键** → Ctrl+Alt+F 唤起输入窗口

═══════════════════════════════════════════════════════════════

❓ 如果还是找不到图标：

   可能原因:
   1. 程序启动失败
   2. 被安全软件拦截
   3. 需要管理员权限

   解决方法:

   【方法1】检查进程是否运行
   
      1. 按 Ctrl + Shift + Esc 打开任务管理器
      2. 在 "进程" 标签中查找 "FlomoQuickNote.exe"
      3. 如果存在，说明程序正在运行
      4. 如果不存在，说明启动失败

   【方法2】重新以管理员身份运行
   
      双击运行: 以管理员身份运行.bat
      
      这会确保:
      ✓ 程序有足够权限运行
      ✓ 全局热键能正常注册
      ✓ 系统托盘图标能正常显示

   【方法3】直接运行 exe 文件
   
      位置: 
      bin\Release\net6.0-windows\win-x64\publish\FlomoQuickNote.exe
      
      右键 → 以管理员身份运行

🎯 快速测试程序是否正常：

   1. 直接按快捷键 Ctrl+Alt+F
      → 如果弹出输入窗口，说明程序正常运行
      → 如果没反应，继续下一步

   2. 打开任务管理器查看进程
      → 如果有 FlomoQuickNote.exe，说明程序在运行
      → 如果没有，说明启动失败

   3. 以管理员身份重新运行
      → 双击: 以管理员身份运行.bat

═══════════════════════════════════════════════════════════════

✅ 找到图标后的第一步：

   1. 右键点击托盘图标
   2. 选择 "⚙️ 设置"
   3. 配置你的 Flomo API 地址
   4. 点击 "保存"
   5. 按 Ctrl+Alt+F 开始使用！
