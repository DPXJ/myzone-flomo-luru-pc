💡 不想安装 .NET SDK？没问题！

═══════════════════════════════════════════════════════════

您有两个选择:

═══════════════════════════════════════════════════════════

选择1: 使用 Python 版本（已经可用）

═══════════════════════════════════════════════════════════

Python 版本虽然有些卡顿，但功能完全可用！

步骤:
  1. 返回项目根目录
  
  2. 双击运行: 一键修复.bat
     (会自动安装 Python 依赖)
  
  3. 双击运行: 后台启动.bat
     (以后台方式启动，关闭窗口不退出)
  
  4. 在系统托盘找图标，右键 → 设置 → 配置API
  
  5. 按快捷键使用

优点:
  ✅ 无需安装 .NET SDK
  ✅ 功能完全一致
  ✅ 立即可用

缺点:
  ⚠️ 设置窗口可能偶尔卡顿
  ⚠️ 体积稍大

───────────────────────────────────────────────────────────

选择2: 安装 .NET SDK 使用 C# 版本

═══════════════════════════════════════════════════════════

C# 版本超级稳定，强烈推荐！

步骤:
  1. 双击运行: 安装NET_SDK.bat
  
  2. 选择自动安装或手动下载
  
  3. 安装完成后关闭并重新打开命令行
  
  4. 双击运行: build.bat
  
  5. 等待编译完成
  
  6. 双击: 以管理员身份运行.bat

优点:
  ✅ 超级稳定，不会卡顿
  ✅ 启动速度快
  ✅ 体积小
  ✅ Windows 原生

缺点:
  ⚠️ 需要先安装 .NET SDK（约 200MB）
  ⚠️ 首次需要编译（1-2分钟）

═══════════════════════════════════════════════════════════

🎯 推荐方案

═══════════════════════════════════════════════════════════

如果您:
  - 只是试用/临时使用 → 用 Python 版本
  - 长期使用，追求稳定 → 装 .NET SDK，用 C# 版本

═══════════════════════════════════════════════════════════

📋 Python 版本快速开始

═══════════════════════════════════════════════════════════

现在就开始（无需安装 .NET）:

第1步: 返回项目根目录
  cd ..

第2步: 安装依赖
  双击: 一键修复.bat

第3步: 启动应用
  双击: 后台启动.bat

第4步: 配置使用
  系统托盘右键 → 设置 → 配置API

完成！

═══════════════════════════════════════════════════════════

📋 C# 版本快速开始

═══════════════════════════════════════════════════════════

第1步: 安装 .NET SDK
  双击: 安装NET_SDK.bat
  选择自动安装

第2步: 重启命令行
  关闭并重新打开命令行

第3步: 编译项目
  双击: build.bat
  等待1-2分钟

第4步: 启动应用
  双击: 以管理员身份运行.bat

第5步: 配置使用
  系统托盘右键 → 设置 → 配置API

完成！享受超级稳定的体验！

❓ 常见问题

═══════════════════════════════════════════════════════════

Q: .NET SDK 安装很大吗？
A: 下载约 180MB，安装后占用约 500MB

Q: 必须装 .NET SDK 吗？
A: 不必须，可以用 Python 版本

Q: Python 版本真的能用吗？
A: 能用！功能完全一致，只是偶尔卡顿

Q: 哪个更好？
A: C# 版本更稳定流畅，但需要先装 SDK
   Python 版本即装即用，但有时卡顿

Q: 我该选哪个？
A: 看您的需求：
   临时用 → Python
   长期用 → C#
