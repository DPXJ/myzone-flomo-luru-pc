🚀 立即开始使用

【步骤1】启动程序

   方式1 (推荐): 双击 "以管理员身份运行.bat"
                → 这样可以确保全局快捷键正常工作

   方式2: 直接运行 exe 文件
         bin\Release\net6.0-windows\win-x64\publish\FlomoQuickNote.exe

【步骤2】找到程序图标

   程序启动后会自动隐藏到系统托盘（屏幕右下角）
   
   👉 点击右下角的 "^" 符号，查看隐藏的图标
   
   找到 "Flomo快速记录" 图标

【步骤3】配置 Flomo API

   1. 右键点击托盘图标
   2. 选择 "⚙️ 设置"
   3. 在设置窗口中输入:
      
      ① Flomo API 地址
         → 格式: https://flomoapp.com/iwh/xxxxx/
         → 如何获取? 见下方说明
      
      ② 全局快捷键 (可选)
         → 默认: Ctrl+Alt+F
         → 可以改成你喜欢的快捷键
      
      ③ 默认标签 (可选)
         → 例如: #灵感 #笔记
   
   4. 点击 "保存"

【步骤4】开始使用

   按快捷键 Ctrl+Alt+F（或你设置的快捷键）
   → 会弹出输入框
   → 输入你的笔记内容
   → 点击 "发送"
   → 笔记自动同步到 Flomo！

📖 如何获取 Flomo API 地址?

   方法1: 在 Flomo 网页版
         1. 打开 https://flomoapp.com/
         2. 登录你的账号
         3. 点击右上角头像 → 实验室
         4. 找到 "API" 或 "输入链接"
         5. 复制完整的 URL

   方法2: 在 Flomo 手机 App
         1. 打开设置
         2. 找到 "实验室" 或 "API"
         3. 开启 API 功能
         4. 复制 API 地址

   ⚠️ API 地址格式示例:
       https://flomoapp.com/iwh/MxxxxxxWxxxxxxxxx/
       
       请确保包含最后的斜杠 "/"

❓ 常见问题

Q: 快捷键按了没反应?
A: 1. 确保程序正在运行（检查系统托盘）
   2. 以管理员身份运行程序
   3. 检查快捷键是否被其他程序占用
   4. 尝试更换其他快捷键组合

Q: 发送笔记失败?
A: 1. 检查 API 地址是否正确
   2. 确保网络连接正常
   3. 检查 Flomo 服务是否正常

Q: 想要更换快捷键?
A: 右键托盘图标 → 设置 → 修改快捷键 → 保存

Q: 如何卸载?
A: 直接删除程序文件即可，不会留下任何垃圾文件
