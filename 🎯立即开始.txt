第 3 步：配置 API 地址
  
  🔑 在系统托盘（右下角）找到图标
  
  步骤：
    1. 右键点击图标
    2. 选择「⚙️ 设置」
    3. 粘贴你的 Flomo API 地址
       （从 flomoapp.com 网站获取）
    4. 点击「测试连接」
    5. 保存
