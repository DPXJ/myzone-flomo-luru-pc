# 🔑 Flomo API 获取教程

本教程将详细介绍如何获取你的Flomo API地址。

## 📝 什么是Flomo API？

Flomo API是Flomo提供的接口，允许第三方应用向你的Flomo账户发送笔记。每个用户都有一个唯一的API地址。

## 🚀 获取步骤

### 步骤1: 访问Flomo官网

打开浏览器，访问：[https://flomoapp.com/](https://flomoapp.com/)

### 步骤2: 登录账户

- 如果已经有账号，点击「登录」
- 如果没有账号，点击「注册」创建一个

### 步骤3: 进入设置页面

登录后，按照以下步骤操作：

1. 点击页面右上角的**头像**或**用户名**
2. 在下拉菜单中选择**「设置」**

![设置入口](https://via.placeholder.com/800x400?text=点击右上角头像→设置)

### 步骤4: 找到API设置

在设置页面中：

1. 在左侧菜单栏找到**「API」**选项
2. 点击进入API设置页面

![API设置](https://via.placeholder.com/800x400?text=左侧菜单→API)

### 步骤5: 获取API地址

在API设置页面：

1. 你会看到一个类似这样的地址：
   ```
   https://flomoapp.com/iwh/MjAxxxxxxxxxxxx/
   ```

2. 点击**「复制」**按钮，或手动选中地址并复制

![复制API](https://via.placeholder.com/800x400?text=复制API地址)

### 步骤6: 配置到应用中

1. 启动「Flomo快速记录」应用
2. 在系统托盘找到应用图标
3. 右键点击图标，选择**「⚙️ 设置」**
4. 在「API地址」输入框中**粘贴**你刚才复制的地址
5. 点击**「测试连接」**按钮验证
6. 如果提示成功，点击**「保存」**

## ✅ 验证API是否有效

### 方法1: 使用应用内测试

在设置窗口中点击「测试连接」按钮：
- ✅ **成功**: 会在你的Flomo中自动创建一条测试笔记
- ❌ **失败**: 检查API地址是否完整复制

### 方法2: 手动测试（可选）

你也可以使用curl命令测试：

```bash
curl -X POST "你的API地址" \
  -H "Content-Type: application/json" \
  -d '{"content": "测试笔记"}'
```

## ⚠️ 安全提示

### API地址安全性

1. **不要分享**: API地址相当于你的密码，不要分享给他人
2. **定期更新**: 如果怀疑泄露，可以在Flomo设置中重新生成
3. **本地存储**: 本应用将API地址加密存储在本地配置文件中

### 重新生成API

如果需要重新生成API地址：

1. 在Flomo网页版「设置」→「API」页面
2. 点击「重新生成」按钮
3. 重新复制新的API地址到应用中

**注意**: 重新生成后，旧的API地址将立即失效

## ❓ 常见问题

### Q1: 找不到API设置在哪里？

**A**: 确保你登录的是Flomo网页版（flomoapp.com），移动App中没有API设置入口。

### Q2: API地址格式不对？

**A**: 正确的格式应该是：
```
https://flomoapp.com/iwh/[一串字母数字]/
```

确保复制了完整的地址，包括开头的`https://`和结尾的`/`

### Q3: 测试连接失败？

**A**: 可能的原因：
1. ❌ API地址复制不完整
2. ❌ 网络连接问题
3. ❌ Flomo服务暂时不可用

**解决方法**:
- 重新复制完整的API地址
- 检查网络连接
- 稍后再试

### Q4: 可以多个设备共用一个API吗？

**A**: 可以！同一个API地址可以在多个设备上使用，所有笔记都会同步到同一个Flomo账户。

### Q5: API有使用限制吗？

**A**: Flomo官方对API调用频率有一定限制，但正常使用不会触及。如果频繁调用可能会被临时限制。

## 📞 获取帮助

如果按照教程仍然无法获取或使用API，可以：

1. **查看Flomo官方帮助**: [https://help.flomoapp.com/](https://help.flomoapp.com/)
2. **联系Flomo客服**: 在Flomo应用内的帮助中心联系客服
3. **检查应用日志**: 查看应用运行时的错误信息

## 🎉 配置完成

配置完成后，你就可以：

1. 按 `Ctrl+Alt+F`（或你自定义的快捷键）
2. 输入笔记内容
3. 一键同步到Flomo

享受快速记录的乐趣！✨
