🎯 推荐启动方式（后台运行）

  双击运行：后台启动.bat
  
  ✅ 优点：
     - 应用在后台运行
     - 关闭命令行窗口不影响使用
     - 在系统托盘常驻
     - 随时按快捷键使用

═══════════════════════════════════════════════════════════

❌ 不推荐的启动方式

  ❌ run.bat           → 会显示命令行窗口
  ❌ 快速启动.bat      → 关闭窗口后程序退出
  ❌ 一键修复.bat      → 仅用于安装依赖

═══════════════════════════════════════════════════════════

📋 完整使用步骤

第1步：安装依赖（首次使用）
  双击：一键修复.bat
  等待安装完成

第2步：启动应用（每次使用）
  双击：后台启动.bat
  本窗口会自动关闭

第3步：配置 API（首次使用）
  1. 在系统托盘（右下角）找到应用图标
  2. 右键点击 → 选择「⚙️ 设置」
  3. 填入 Flomo API 地址
  4. 点击「测试连接」
  5. 保存

第4步：开始使用
  按 Ctrl+Alt+F → 输入内容 → Ctrl+Enter

═══════════════════════════════════════════════════════════

🔍 如何找到系统托盘图标

系统托盘位置：屏幕右下角，时间旁边

如果看不到图标：
  1. 点击系统托盘的「^」按钮
  2. 展开隐藏图标
  3. 找到 Flomo 快速记录图标

═══════════════════════════════════════════════════════════

⚙️ 如何配置 API 地址

详细说明请看：如何配置API.txt

快速步骤：
  1. 访问 https://flomoapp.com/
  2. 登录 → 设置 → API
  3. 复制 API 地址
  4. 在应用设置中粘贴

═══════════════════════════════════════════════════════════

✨ 启动后可以做什么

1. 按 Ctrl+Alt+F 唤起输入窗口
2. 右键系统托盘图标查看菜单
3. 在设置中自定义快捷键和标签

═══════════════════════════════════════════════════════════

⚠️ 注意事项

1. 使用「后台启动.bat」启动后，即使关闭启动窗口，
   应用仍在后台运行

2. 如需退出应用：
   右键系统托盘图标 → 选择「退出」

3. 建议将「后台启动.bat」创建快捷方式到桌面

═══════════════════════════════════════════════════════════

问题：启动后没看到任何窗口？

这是正常的！应用在后台运行：
  ✅ 查看系统托盘是否有图标
  ✅ 按 Ctrl+Alt+F 测试是否能唤起窗口
  ✅ 如果都正常，说明启动成功
