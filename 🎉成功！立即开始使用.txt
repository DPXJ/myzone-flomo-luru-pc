🚀 立即开始使用（三步）：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步：启动程序

   双击运行: CSharp版本\以管理员身份运行.bat
   
   （或直接双击 exe 文件）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第2步：找到托盘图标

   • 看屏幕右下角（时间旁边）
   • 点击向上的箭头 "^" 或 "∧"
   • 找到 "Flomo快速记录" 图标
   
   ⚠️ 重要: 不在任务栏，而是在系统托盘！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第3步：配置 API

   • 右键点击托盘图标
   • 选择 "⚙️ 设置"
   • 输入 Flomo API 地址
   • 点击 "保存"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 开始记录笔记：

   按快捷键: Ctrl + Alt + F
   
   → 弹出输入窗口
   → 输入你的笔记内容
   → 点击 "发送"
   → 自动同步到 Flomo！

═══════════════════════════════════════════════════════════════

📖 如何获取 Flomo API：

   1. 访问: https://flomoapp.com/
   2. 登录你的账号
   3. 点击头像 → 实验室 → API
   4. 复制完整的 API 地址
   
   格式示例: https://flomoapp.com/iwh/xxxxx/

          → 查找 "Flomo快速记录"

❓ 常见问题：

   Q: 快捷键没反应？
   A: 1. 确保程序正在运行（查看任务管理器）
      2. 以管理员身份运行程序
      3. 尝试更换其他快捷键（在设置中修改）

   Q: 找不到托盘图标？
   A: 1. 点击右下角 "^" 展开隐藏图标
      2. 可能在隐藏区域
      3. 或者程序没有成功启动（查看任务管理器）

   Q: 发送失败？
   A: 1. 检查 API 地址是否正确
      2. 检查网络连接
      3. 确保 API 地址包含最后的 /

   Q: 如何退出程序？
   A: 右键托盘图标 → 选择 "🚪 退出"

   Q: 如何开机自动启动？
   A: 1. 按 Win+R 输入: shell:startup
      2. 创建程序的快捷方式到打开的文件夹
      3. 设置快捷方式以管理员身份运行
